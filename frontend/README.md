# Gmail Lead Sync - Frontend

React + TypeScript + Vite frontend for the Gmail Lead Sync Admin Dashboard.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy environment variables:
```bash
cp .env.example .env
```

3. Start development server:
```bash
npm run dev
```

The app will be available at http://localhost:5173

## Build

```bash
npm run build
```

The production build will be in the `dist/` directory.

## Testing

```bash
npm test
```

## Tech Stack

- React 18
- TypeScript 5
- Vite 5
- Tailwind CSS
- React Router
- Axios
- React Hook Form + Zod
